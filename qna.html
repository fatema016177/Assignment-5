What is the difference between innerHTML, innerText, and textContent ?
innerHTML All the elements inside a div can be found with html tags, innerText can only find the inner text. And textContent gives all the text inside a div only without html tags and any style related code
What is event delegation in the DOM?
Event Delegation is a form of event handling, where an event listener is set on its parent element instead of a child element. This makes the event work correctly even though new elements are added dynamically.
What is event bubbling in the DOM?
Event Bubbling is an event propagation mechanism, where an event first occurs on the element that is clicked and then propagates step by step to its parent and upper elements.
